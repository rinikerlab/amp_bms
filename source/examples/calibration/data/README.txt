This is a collection of files that were used for calibration of ML/MM interactions

'HFE_molecules_pdbs' - collection of PDB files of 25 molecules from FreeSolv that were used
 for Hydration Free Energy (HFE) calculations

'default_csv_parameters.json' - default configuration file for OpenMM CSV reporter

'ff_default_tip4pfb.json' - default configuration file for OpenMM ForceField if TIP4P-FB water model is used

'forcefield_parameters_HFE_molecules.json' - file containing OpenFF-2.2.0 parameters for 25 molecules used for HFE calculations

'forcefield_water_off.json' - file containing OpenFF-2.2.0 parameters for water in ML zone used for ML-water-in-MM-water simulations

'input_water.pdb' - PDB file of ML water used for ML-water-in-MM-water simulations

'MIN_state_dict' - weights of AMP model

'molecules_for_HFE.txt' - file with SMILES of 25 molecules used for HFE calculations

'molecules_water_in_water.txt' - file with water SMILES used for ML-water-in-MM-water simulations

'PARAMETERS_MIN_LRv2_tip4pfb.yaml' - AMP configuration file for use with TIP4P-FB water model
 (polarization scaling set to optimal 0.35)

'HFE_mols_FreeSolv_conversion.json' - JSON file used to convert the internally used enumeration of 25
 molecules used for HFE calculations to the IDs in FreeSolv database
 
'Trp_cage.pdb' - PDB file of Tryptophan cage (not solvated) used for Trp-cage simulations for ML/MM calibration
